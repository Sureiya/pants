Pants Documentation
*******************

*A lightweight framework for writing asynchronous network applications in
Python.*

Install Pants using `pip <http://pypi.python.org/pypi/pip>`_::

    pip install pants

Using `setuptools <http://pypi.python.org/pypi/setuptools>`_::

    easy_install pants

Or clone the `git <http://git-scm.com/>`_ repository::

    git clone git://github.com/ecdavis/Pants pants

Pants requires `Python 2.6+ <http://python.org/>`_ - Python 3 is not yet supported.
